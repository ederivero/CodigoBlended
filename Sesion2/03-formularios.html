Lista de Elementos

Nro Nombre Elemento Descripción del Elemento
1 input Es un elemento de entrada en que podemos ingresar datos
2 textarea Es un input para que podamos ingresar una mayor cantidad de contenido, en más de una fila
3 select
  • Select es un input que muestra una lista de opciones
  • Y el usuario podrá elegir solo una de esas opciones (un elemento option)
4 button Este elemento representará un botón que puede ejecutar una acción
5 label Va a ser un texto que va a estar asociado a un input
6 fieldset Es un elemento que me agrupa varios elementos de un formulario
7 legend
  • Es un elemento que se comporta como título del fielset
  • Da mayor información

Tipos de Inputs

Nro Nombre Input Descripción del Input
1 text
  1. Es un input que sirve para ingresar texto plano y visible
  2. Por Ej.: Para Nombres, Apellidos y Direcciones
2 number
  1. Es un input solamente para números
  2. Sirve para cantidades, Dinero, telefonos, edad
3 email
  1. Input para ingresar solamente correos electrónicos
  2. Para la validación de que el texto que coloquemos dentro de este input tenga una @
4 date
  1. Es un tipo que me permite ingresar fechas, de forma amigable
  2. Me permite añadir de forma amigable, la fecha con día/mes/año
5 password
  1. Va servir para ingresar nuestra contraseña sin que se muestre el texto
  2. Mostrará Puntos en reemplazo de los caracteres del Password
6 time
  1. Me permite ingresar de forma amigable, horas, minutos y segundos
7 color
  1. Me permite ingresar un color, con la ayuda de una paleta de colores
8 checkbox
  1. Me permite seleccionar varios elementos
  2. Una lista de compra por ejemplo
9 radio
  1. De varios elementos solo me permite seleccionar uno de esos elementos
10 file
    Me permite subir archivos a un servidor
11 reset
  • Input en forma de un botón.
  • Sirve para limpiar todos los campos del formulario en el que se encuentre
12 submit
  • Input en forma de un boton
  • Sirve para enviar TODOS los datos del formulario al servidor
